# qb-web

## Project setup
node: v18.20.8

需要先安装yarn 3.5.0版本，建议使用nvm。

```
npm install corepack -g
corepack enable
corepack corepack prepare yarn@3.5.0 --activate
yarn install
```

### Compiles and hot-reloads for development
```
yarn run serve
```

### Compiles and minifies for production
```
yarn run build
```

### Lints and fixes files
```
yarn run lint
```
